# Youtube speed control (mkr fork)

A simple way to control the speed of YouTube playback in Firefox.

> This is a Manifest V3 fork of [Toxblh/youtube-speed-control](https://github.com/Toxblh/youtube-speed-control). All credit for the original idea and implementation goes to **[@Toxblh](https://github.com/Toxblh)** and contributors. Licensed under the MIT License (see `LICENSE`).

![](src/option/YouTubeSpeedOption.jpg)

Control the speed of YouTube playback using your keyboard:
- Press `+` to increase and `-` to decrease the playback speed.
- Or hold `Shift` + mouse wheel.

This plugin doesn't work together with "Enhancer for YouTube™".

## Changes in this fork
- **Native player controls**: speed indicator can render inside the YouTube player UI instead of as an overlay.
- **Preset speeds**: configurable list (e.g. `1.5, 2`) shown as one-click buttons.
- **Manifest V3** (Firefox 142+).
